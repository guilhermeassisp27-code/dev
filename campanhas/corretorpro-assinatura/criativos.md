**Checagem de macrovariação (mínimo 2 níveis diferentes entre QUALQUER par):**
- 1 vs 2: mecânico (vídeo→imagem) + visual (pessoa/carro → sem pessoa/objetos) ✓
- 2 vs 3: mecânico (estática→carrossel) + visual (objetos→mockups+pessoa) ✓
- 3 vs 4: mecânico (carrossel→vídeo dinâmico) + visual (escritório→apê) ✓
- 4 vs 5: mecânico (vídeo→imagem) + proporção (9:16 igual, mas) + visual (pessoa→sem pessoa) + temático (desejo→prova) ✓
- 5 vs 6: mecânico (imagem→vídeo) + proporção (9:16→16:9) + temático (prova calma→urgência agressiva) ✓
- 1 vs 6: mecânico (plano único→split screen) + proporção (9:16→16:9) + temático (medo calmo→urgência agressiva) ✓
- 2 vs 5: ambos sem pessoa em cena, mas mecânico (estática 1:1→estática 9:16) + temático (reflexivo/identificação→prova social objetiva) + nível de consciência (P1→P3, copy e CTA completamente diferentes) ✓

Nenhum par da bateria repete formato + proporção + visual + tema simultaneamente — todos diferem em pelo menos dois níveis, conforme exigido pela base de conhecimento (proteção contra Andrômeda).
