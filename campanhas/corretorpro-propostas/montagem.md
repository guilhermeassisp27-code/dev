# Montagem dos vídeos — CorretorPRO (clipes de 8s do Sora + edição)

Fluxo: gerar cada clipe no Sora (prompts no super-prompts.md, seção "Vídeos 8s") →
importar na ordem no CapCut/Canva → aplicar os overlays cronometrados abaixo →
música baixa (10-15%) → legenda automática. Gancho tem que bater nos primeiros 3s.

Dica de montagem:
- Corte ~0,3-0,5s do início de cada clipe (Sora abre parado).
- Overlay = texto grande por cima do vídeo, reforça quem assiste sem som.
- Mantenha a fala do Sora; música só de fundo.
